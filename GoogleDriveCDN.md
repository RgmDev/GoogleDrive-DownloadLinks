### Google Drive CDN (Php)
You can use a file stored in Google Drive like a CDN 

PHP Example:

CSS file stored in Google drive (style.css)
```
body {
  background-color: rgb( 171, 202, 252 );
  padding: 20px 50px 20px 50px;
}

h1, h2, p {
  font-family: arial;
  color: darkblue;
  text-align: center;
}

h1 {
  font-size: 40px;
}

h2 {
  font-size: 24px;
}

p {
  text-align: left;
}
```
